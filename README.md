# Flash Cámaras

Sitio web del taller de servicio técnico Flash Cámaras (Medellín, Colombia). Mantenimiento, reparación y accesorios para cámaras Sony, Canon, Nikon, Fuji.

## Stack

- React 19 + Vite
- Tailwind CSS 3
- React Router 7
- Framer Motion
- Lucide React (iconos)

## Desarrollo

```bash
cd web
npm install
npm run dev
```

## Build de producción

```bash
cd web
npm run build
```

El output queda en `web/dist/`.

## Deploy

Desplegado en Vercel: <https://flash-camaras.vercel.app>

```bash
cd web
npx vercel --prod
```
